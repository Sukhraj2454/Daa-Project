Travelling Salesman

Given a list of cities and the distances between each pair of cities, what is the shortest possible route that visits each city once and returns to the origin city?

Complexity:O(n^2)

Prim's Minimum Spanning Tree

Prim's (also known as Jarn√≠k's) algorithm is a greedy algorithm that finds a minimum spanning tree for a weighted undirected graph.
The algorithm operates by building this tree one vertex at a time, from an arbitrary starting vertex, at each step adding the cheapest possible connection from the tree to another vertex.

Complexity:O ((|V|+|E|)log|V|)(Where |V|is the number of vertices and |E| is the number of edges.)

Kruskal's Minimum Spanning Tree

Kruskal's algorithm is a minimum-spanning-tree algorithm which finds an edge of the least possible weight that connects any two trees in the forest. It is a greedy algorithm in graph theory as it finds a minimum spanning tree for a connected weighted graph adding increasing cost arcs at each step.

Complexity:O (|E|log|V|)(Where |V|is the number of vertices and |E| is the number of edges.)

Dijkstra's Shortest Path

Dijkstra's algorithm is an algorithm for finding the shortest paths between nodes in a graph. Dijkstra's algorithm uses a data structure for storing and querying partial solutions sorted by distance from the start. While the original algorithm uses a min-priority queue.

Complexity:O(|V|+|E|log |V|)(Where |V|is the number of vertices and |E| is the number of edges.)

0/1 Knapsack

A thief robbing a store finds n items. The ith item is worth 'k' dollars and weighs w(i) pounds, where k and w(i) are integers. The thief wants to take as valuable a load as possible, but he can carry at most W pounds in his knapsack, for some integer w. Which items should he take?

Complexity:O(nW) where, n is the number of items and W is the capacity of knapsack.
